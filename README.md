## Tech Stack

NextJs, Tailwind CSS

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/rohitpotato/easy-search.git
```

Go to the project directory

```bash
  cd easy-search
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
